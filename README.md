Reservación de Viajes  con JAVA + POO.

Aplicación de consola para practicar Programación Orientada a Objetos.

Uso de NEATBEANS.


# Cómo se ejecuta 
1. Abre en NetBeans
2. Ejecuta `Reservacion_viajes.java`
3. Usa el menú en consola

## Lo que aprendí
- Clases y objetos
- Encapsulamiento (`private` + getters/setters)
- Lógica de reservas
